<br/>    雪山老君道：“陈明道友未免太过担心了，不过一个悟道境而已，能够掀起多大的风浪？”

<br/>        剑王阁老君也是道：“就算是从入道境突破到了悟道境，修为提升很快，但是不到大君，如何能够影响大局？”

<br/>        没了陈明，找谁炼制阴阳和合丹去！
<br/>    
<br/>        这根大腿这么粗，一定要抱住！
<br/>

<br/>        剑王阁老君也是道：“燕山数万炼丹师，只是现在，看门下弟子如今恨不得为陈明去杀上一场的模样，就知道将来燕山人脉之广，群星老东西，你倒是打得一手好算盘！”

<br/>        铁衣老君却是道：“不过我看那陈明，似乎并没有对紫霞倾心的意思。”
<br/>    
<br/>        群星老君哈哈大笑道：“女追男，加上又是两位新晋大君，彼此惺惺相惜，共同经历患难，能够有多难，大不了，我再在两人的封号上做一些手段，推波助澜一番！”

<br/>        陈明却是不知道四位老君那边正在乱点鸳鸯谱，带着卓青瑶，火急火燎地就往燕山去，卓青瑶道：“不知道燕山出了什么事，三师弟的修为，增长这么迅速，一定是出了大事了。”

<br/>        陈明道：“现在管不了那么多了，我必须先回去看看。如果真的是妖族大君动手了，我必定将他碎尸万段！”

<br/>        两人风风火火赶回燕山，已经是数日之后，但是陈明回到沙海之中时，却是没有看到半点战争的景象，“嗯，这是怎么回事？”
<br/>    
<br/>        卓青瑶道：“好像并没有爆发战争。”

<br/>        陈明道：“回燕山看看。”
