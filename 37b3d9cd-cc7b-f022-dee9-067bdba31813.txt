　　第572章
<br/>　　孟玄青身后立着一只木笼，笼内装着两个人，自然就是千里赶到东海的李莫愁和李婉儿。
<br/>　　她们现在都处于昏睡中，这是某种药物导致的，对一代巫医孟玄青而言，这并不是太困难的事情。
<br/>　　他来到这座岛已经有几天，将全权负责组织这场对程大雷的伏击。这对眼高于顶的书院弟子来说，并不是一件令人愉悦的事情。于是，在过去的几天时间内，不断有人无故寻衅以达到对他挑战的目的。结果……却是无一例外的败北。
<br/>　　围棋、经学、剑道、占星、卜卦……当然也包括医术。这个已经老到看不出多大年纪，仿佛一阵风就能吹倒的老人，展现了完全碾压的实力，让书院弟子低下高贵的头颅，心服口服。
<br/>　　孟玄青渐渐找出些当年风流倜傥的味道，胸中也添了几分豪情。因这份豪情在，他看上去年轻了一些。
<br/>　　今天已是约定的日子，在孟玄青的组织下，岛上的布局已经完成。只等着程大雷自投罗网，然后关门打狗。
<br/>　　“来了。”
<br/>　　不知是谁喊了一嗓子，立在高处的孟玄青向西望去，只见远远的，一叶扁舟顺风而来。
<br/>　　船头立着程大雷以及一头黑牛。
<br/>　　“为什么要骑牛上岛呢？”
<br/>　　孟玄青奇怪的皱起眉头。年纪大了，有些事情他搞不懂，其实也不太想搞懂，他的目的只是杀了程大雷。
<br/>　　今时今日，天高地阔，是杀人的好天气。
<br/>　　乘舟东来，程大雷看到了白沙岛，也看到了岛上来来回回的黑影。知道有很多人在等自己，此刻程大雷有种明知山有虎偏向虎山行的感觉。
<br/>　　知道很多人在等自己，但他还是来了。
<br/>　　昂
<br/>　　黑牛抬起头，发出一声长哞。还未登上海岛，程大雷已经遇到了第一场阻击。
<br/>　　大贼张六。
<br/>　　张六今天没有带棍，而是用了锤子和凿子。作为一个职业海贼，他拥有超凡的水性，能在水下闭息良久。
<br/>　　他隐在海水下，当船经过时，用工具凿穿船底。假若程大雷的本事连张六这一关都过不去，那张六不介意直接杀掉他。剩下的时间，大家可以在海滩上搞个篝火晚会什么的，反正今天天气不错。
<br/>　　小舟的船底被凿穿，海水咕咚咚涌进来，风已吹不动船，黑牛不安的发出哞哞声。
<br/>　　程大雷纵然有些水性，可也不能和张六相提并论。他自然觉察到了危险，但没有太多动作，只是将手搁在剑柄上，微微眯起眼睛。
<br/>　　如一张拉满的弓，任何试探性的触碰，就会令他开弓走箭。
<br/>　　水面起伏，游鱼从船边浮过，一切都如这大海一般，粗看并无太多奇特，但仔细体味，却会发现它深不可测，令人恐怖。
<br/>　　咕咚咕咚……
<br/>　　水底冒出气泡。
<br/>　　程大雷突然暴起，如拉满的弓射出剑，他身子跃起，剑在空中出鞘，然后从水底撩过。
<br/>　　空中一招猴子捞月，程大雷又返回到小舟，看上去像是什么都没有发生，但匹夫剑的剑尖，已经淅沥沥往下滴血。
<br/>　　张六从水中冒了出来，他脸上出现一道可怖的伤口，一双凶恶的眼神盯着程大雷。
<br/>　　张六可以在海水中闭息良久，但他终究不是鱼，待得再久，也需要浮到水面上换气。
<br/>　　却是没想到，这一闪而逝的机会，还是被程大雷抓到了。然后，程大雷干净利落的给了他点教训。
<br/>　　程大雷直视着他的目光，晃了晃肩膀，扬起下巴，做出挑衅的眼神。
<br/>　　张六怒不可遏，但程大雷的一剑的确令他忌惮。作为一个贼，能好端端的活这么多年，若以为他只有武勇而没有脑子，那就大错特错了。
<br/>　　他不来，程大雷过去。
<br/>　　他从船上丢出几块被砍碎的木板，身子腾空而起，在木板落水的刹那，脚尖在上面一点，身子再次向前突进，不断接近张六。
<br/>　　张六大骇，程大雷这一手轻功当真漂亮到极致，令其不寒而栗。几乎没有任何考虑，他一个猛子扎进海水中。
<br/>　　程大雷的剑擦着他的头顶而过。
<br/>　　等张六再从海水中浮上来时，还感觉头顶嗖嗖冒凉气。这个时候，程大雷已经借轻身功夫落到岸边，当然，那艘小舟也已沉没。墨玉乌月牛浮水而过，湿漉漉的立在程大雷身边。
<br/>　　它晃了晃脑袋，将水珠甩尽，傲慢的昂起头，又丑又凶的牛头，颇有种睥睨四方的感觉。
<br/>　　果然出自蛤蟆城，有种蠢强蠢强的气质。
<br/>　　程大雷盯着水中的张六，冲他招了招手。
<br/>　　一股怒火从张六心底冒起，程大雷的确很强，俨然已是天下第一贼。可我当年也是海上大枭，难道你就能小看我么。这里是海上，而且我资历还比你老，你怎么可以小看我呢。
<br/>　　他身子一点一点浮起，最后只有两条腿压在水里。脚底踏水，宛若被风吹到岸边，身子一接触陆地就跃起，手中的凿子锤子向程大雷砍来。
<br/>　　的确很凶，倒是不能小觑。他和程大雷的打法不同，依仗的是一身蛮力，快速强攻。程大雷不敢也不能迎接，一退再退，避过张猛的锋芒。
<br/>　　这样的强攻，撑不了太久的，只要他锋芒耗尽，就是程大雷出剑的时刻，也是收割对方性命的时候。
<br/>　　“六爷，接棍。”
<br/>　　宋寒之此刻到了，将大棍隔空抛来。张六寻机接住，趁手的兵器再手，他变得更加凶狠。
<br/>　　于此同时，宋寒之的剑已经攻了过来，棍中夹剑，当真令程大雷难以阻挡。
<br/>　　需要一开始就打这么急么，难道不是该双方亮明阵仗，互相放几句狠话再动手么。
<br/>　　对方攻得急，程大雷不可不接，他接得很狼狈。
<br/>　　他忽地骑上黑牛，口中呼嘞嘞喝了一声，黑牛四蹄放开，撞向张六和宋寒之的联手。
<br/>　　张六和宋寒之都挡不住这黑牛，不得不撤开。
<br/>　　程大雷自然不能和二人缠斗，他已经会过书山的剑阵，了解它的威力。上次不过是二十一柄剑，程大雷就捉襟见肘，更何况今天这么大阵仗。
<br/>　　所以，程大雷的战略很简单，向骑着黑牛趟一遍，把对方准备的手段都趟出来再说。
