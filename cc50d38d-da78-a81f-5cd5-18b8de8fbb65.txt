<br/>    陈明朝着尸九泉呵呵一笑，“抱歉，这棵树我早就已经到手了，你算算我灭杀你第一个分身，到现在的时间，你觉得，我没有这棵树，我做得到吗？”
<br/>    
<br/>        尸九泉的双眼出现一片灰色，万念俱灰，“原来你，早就已经得到了这棵树！”
<br/>    
<br/>        陈明将手中尸九泉的脑袋当做球扔了扔，然后看向已经封闭的仙路大门，然后认真道:“话说，我当初觉得和你一战可能打不赢，所以断绝了自己的回去的路，那么问题来了，我现在要怎么回去呢？你那几个手下，应该还是会按照你的意思打开这扇门吧？”
<br/>    
<br/>        尸九泉:“我的眼睛！”
<br/>    
<br/>        陈明赶紧将自己的指头从尸九泉的眼眶之中出来，“哦哦哦，对不起，没注意。”
<br/>    
<br/>        尸九泉继续道:“你要是放我一条生路，我就让那些活尸全部重新死去。这样，整个大陆的人口也不会因为这一次减少太多，这样一来，你每年都会有一定数量的灵魂，不然的话，你以后每年得到的灵魂数量，将会大减！”
<br/>    
<br/>        陈明反手就是一把将尸九泉的脑袋摁碎在地面上，“我从不和敌人做交易，你犯下的罪孽，太重了，唯有死才可以偿还，而且，不是因为你，我燕山弟子，可不会死去那么多！”
<br/>    
<br/>        “犯我燕山者，虽远必诛，虽强必战！”
<br/>    
<br/>        就在这时，陈明忽然接受到了系统小姐姐的声音，“你的亲传弟子，卓青瑶，已经突破到了超脱，你获得了一百万点师德！”
<br/>    
<br/>        “哦吼，这个小丫头，终于是突破到了超脱了，她现在的实力，堪比君主吧？”
<br/>    
<br/>        陈明背靠着仙路的大门坐下来，看着前方没有任何生灵存在的仙路，“早知道，就把尸九泉留下来了，还可以有个人聊天，现在，还真是无聊啊！”
<br/>    
<br/>        一个月后。
<br/>    
<br/>        在一个月之前，所有的活尸如同是失去了意识一般，重新跌落，众多古仙门，君主级宗门惊讶之时，同样也是将仙路的钥匙抢了回来。
<br/>    
<br/>        虽然不知道发生了什么事，但是还是吃惊无比。
<br/>    
<br/>        众多君主，联合燕山四位宗主，前往玉虚宫察看情况。
<br/>    
<br/>        他们也想知道所以然，在处理了一个月之后，诸位君主，和燕山的四位宗主终于是来到了仙路大门前，众人来到此处，朱雀也是把陈明进入仙路和鬼仙搏命的消息告诉了他们。
<br/>    
<br/>        朱雀最后看向燕山的四位宗主，道:“你们师父留下的最后一句话是:告诉你们，他爱你们！”
<br/>    
<br/>        卓青瑶冷笑了一声，“这个老东西，一直让我们不要煽情，自己到头来，却是最煽情的一个！”
<br/>    
<br/>        凌仙看向拿着钥匙的四人，认真道:“既然活尸已经失去了意识，那就说明鬼仙已经死了，你们还不打开仙路将师父放回来？”
<br/>    
<br/>        紫霞仙子如今乃是群星仙门的宗主，靠着和燕山的关系交好，紫霞仙子登上了群星厦门宗主之位。
<br/>    
<br/>        紫霞仙子也是道:“不错，如今活尸都已经死去了，一定是陈仙师胜了，你们还不打开仙路，放陈仙师回来？”
<br/>    
<br/>        紫霞仙子走到卓青瑶的面前，将自己的钥匙递给了卓青瑶。
<br/>    
<br/>        卓青瑶看了一眼，并没有直接接下，而是看向其他三位古仙门的宗主。
<br/>    
<br/>        雪山宗宗主道:“如今鬼仙和盟主大战，我们并不知道结果，如果说是鬼仙胜了，为了营造盟主胜了的假象呢？”
<br/>    
<br/>        剑王阁的阁主也是道:“不错，若是鬼仙胜了，仙路已经彻底关闭，他也没有任何机会可以过来，唯有让这些活尸失去意识，营造出这个假象，我们才会打开仙路大门，这样一来，鬼仙不就再次降临，鬼仙降临，我们可是没有任何机会胜利的！”
<br/>    
<br/>        李素衣的眼中开始泛红，大怒道:“可笑，若是鬼仙赢了，他完全可以操控这些活尸来抢夺钥匙，仙人的活尸，可是有着仙之下的实力，岂是我们能够抵挡住的？”
<br/>    
<br/>        铁牛谷谷主道:“哼！不然，仙路大门已经关闭，鬼仙已经失去了和这一座大陆沟通的机会，如何能够操控这些活尸，这些活尸必是因为仙路大门关闭无法操控活尸！”
<br/>    
<br/>        陈灵儿此刻也是忍不住了，“你们，根本就是不想要放师父回来！”
<br/>    
<br/>        其他四位君主，看现在的情况不太妙，重瞳子道:“我们几个没有钥匙，就不掺和了，你们自己商量吧。”
<br/>    
<br/>        铁牛谷的谷主笑道:“我们岂是不想要放你们师父回来，你们燕山四位宗主天赋绝绝，只要达到仙人境界，那不就可以顺利地打开仙路了吗？无论是盟主胜了，还是鬼仙胜了，都没事，不过几百年而已！”
<br/>    
<br/>        凌仙怒道:“仙路不开，如何突破到仙人境界？而且，师父如今只是悟道境，他只有三百年的寿元！”
<br/>    
<br/>        雪山宗的宗主道:“那也至少要等这片大陆之上出现足够多的仙之下，才有机会对抗一位仙人，不然，若是鬼仙活着，后果不堪设想！”
<br/>    
<br/>        卓青瑶一言不发地看着众人争吵，李素衣，凌仙，陈灵儿还想继续说理，卓青瑶却是挥手道:“住嘴！他们说的有道理，我们回燕山。”
<br/>    
<br/>        三个亲传弟子看向卓青瑶，若不是大师姐发话，他们绝对不会就此停止。
<br/>    
<br/>        卓青瑶转身离去，三个亲传弟子也是跟着卓青瑶离去。
<br/>    
<br/>        紫霞仙子看向卓青瑶离去的身影，有些怒气地扫了众人一眼，追向了卓青瑶。
<br/>    
<br/>        待得紫霞仙子和燕山四位宗主离开，铁牛谷的谷主道:“不知道，到底是盟主胜了，还是鬼仙胜了。”
<br/>    
<br/>        雪山宗的宗主微微一笑，“这重要吗？陈明胜了，那么陈明便是仙人实力，我们必将屈服于他的淫威之下，若是鬼仙，整个大陆变成鬼界，无论是陈明还是鬼仙活着，都没有什么区别，而现在，便是最好的结果，我们依旧，掌控的整个大陆的半壁江山！”
<br/>    
<br/>        剑王阁的阁主道:“若不是因为现在燕山有着救世的美名，我刚刚就直接动手了！”
