　　“咔咔~”
<br/>　　一阵骨骼摩擦的声音响起，两人惊悚的注视着尸体，小胖捏着唐刀的手心，满是汗水。
<br/>　　“一百分，一百分……”
<br/>　　一道童音响起，门框上缓缓出现了一个拉长脖子的头颅，两根暗棕色的马尾，在脑后慢慢晃动着。
<br/>　　像是皮筋一般，尸体的脖子突然拉长，将头颅伸到了小胖面前，肮脏的口水，顺着嘴角滑落在地面上。
<br/>　　“我去，什么鬼东西！”
<br/>　　小胖眼神有些惊慌，握着唐刀的手掌向上一撩，轻飘飘的将头颅顺着脖子斩下，瞬间，令人心烦意乱的声音彻底消失。
<br/>　　“诶嘿嘿，小子不错啊，三两下就解决了一个尸兄，这么能打，要不要……”
<br/>　　刺耳的声音再度从扬声器中响起，小胖微微喘了一口粗气，转头劈碎了墙角的扬声器。
<br/>　　声音戛然而止，楼道恢复了先前的安静。
<br/>　　“还以为是什么东西，吓死胖爷我了。”
<br/>　　抹去唐刀上面沾染的血液，小胖声音有些颤抖，口罩外面裸露的皮肤，有些苍白。
<br/>　　拍了拍小胖的肩膀，刘哔有些奇怪，小胖现在，和他第一次杀人之后的那种样子差不许多。
<br/>　　可这家伙，刚才杀的明明只是一头尸兄啊。
<br/>　　“我没事，走吧，去找你哥哥。”
<br/>　　小胖握了握手中的刀柄，缓缓推开了步行楼梯的大门。
<br/>　　……
<br/>　　“哎，你看看这里。”
<br/>　　两人走到外面的走廊上，才发现之前那个尸兄，是被人将脖子绑在了天花板上的一个铁钩上，出于好奇，小胖走近查看了一下。
<br/>　　听到小胖的呼唤，刘哔顺着小胖手指的方向看去，发现了尸体上的胸牌。
<br/>　　精神科病人-01号。
<br/>　　是这个医院的病人么，但为什么变成尸兄了还会被人操控，小胖停下脚步，眉头皱成一团。
<br/>　　他感觉这个地方越来越危险了。
<br/>　　“你说……我哥哥会不会也变成了这样。”
<br/>　　眉宇间含着担忧的神色，刘哔上前走了几步，注视着尸体身上的胸牌。
<br/>　　是在这里的吗，不对，但好像真的有这么一个胸牌。
<br/>　　剧情有些记不清了，但应该没有出错，那个博士，好像还在这里。
<br/>　　这样说的话，主角现在还没有来过？？
<br/>　　太好了。
<br/>　　那样，他就不必再花费精力寻找别的方式了……
<br/>　　“你哥哥也是神经病吗？”小胖转过头来，下意识的脱口而出，随后意识到什么，连忙解释道：“额，我是说这里的病人。”
<br/>　　小胖的表情有些尴尬，但刘哔根本没有在意，依旧注视着胸牌，不知道在想些什么。
<br/>　　“放心，你哥哥会没事的，我们一定能找到他。”拍了拍刘哔的肩膀，小胖出声安慰道，随后，像是发现了什么，疑惑的看了看刘哔的双眼：“咦，哭了么，眼都红了，唉，放心，肯定会没事的。”
<br/>　　勾了勾嘴角，刘哔笑容有些僵硬，却也没多说什么，两人顺着楼层，开始一间屋子一间屋子的寻找起来。
<br/>　　他不知道博士具体在哪个房间，但最终肯定会寻找到的。
<br/>　　以博士那种扭曲的性格，肯定不会离开，毕竟，他们在博士眼中，也只是两个进行人体实验的小白鼠而已。
<br/>　　“哒……哒……”
<br/>　　沉闷的脚步声在走廊里回荡着，突然又戛然而止，小胖停住了脚步，似乎是发现了什么。
<br/>　　“怎么了？是不是……”
<br/>　　刘哔有些疑惑的开口，看到小胖噤声的手势，吞下了还没说完的话语。
<br/>　　“你听。”
<br/>　　小胖指了指前方的一个房间，刘哔竖耳听去，注意到一股极为细微的摩擦声。
<br/>　　有些像，衣服在门框上摩擦的那种声音。
<br/>　　似乎，里面存在着什么东西。
<br/>　　“诶嘿嘿哈哈哈……被发现了呢。”
<br/>　　有些病态的声音再度响起，两人扭头，看到了墙上悬挂着的扬声器。
<br/>　　“这两个也是我的杰作，好好……享用吧，诶嘿嘿嘿……”
<br/>　　狂躁的音乐声再度响起，病房门被猛地破开，从中涌出数只身穿病号服的尸兄，相同的是，这些尸兄都带着同款的黄色帽子。
<br/>　　小胖目光一沉，挥刀劈碎了墙壁上的扬声器，这些尸兄，似乎是靠音乐控制，他在想，如果将音乐制止，是否能解开现在的困境。
<br/>　　事实上他的想法没有出错，但走廊里，可不止这一个扬声器。
<br/>　　即使这一个被劈碎了，但音乐的声音还是陆续传来，而此时，那些尸兄们已经将两人围住，根本不给他们寻找隐藏扬声器的机会。
<br/>　　“打扫，打扫……”
<br/>　　“扣分，扣分！”
<br/>　　尸兄们口中传出含糊不清的嘶吼声，将两人团团围住，几颗拉长脖子的头颅，同时向小胖飞来。
<br/>　　奇怪的是，这些尸兄对刘哔都没有产生攻击欲望，他们的目标，不约而同的放在了小胖身上。
<br/>　　“乒！”
<br/>　　金属碰撞的声音响起，刘哔脸色苍白，看着手中破碎的菜刀，脸上露出不可置信的神情。
<br/>　　而被他攻击到脖颈处的尸兄，根本没受到任何影响，唯一留下的痕迹，就是一道微不可查的白痕。
<br/>　　这脖子，竟然比石头还要坚硬！
<br/>　　与此同时，小胖那边的战斗也彻底打响，面对这些勇猛异常的尸兄，小胖手中的唐刀，也不再像先前那样锋利。
<br/>　　他的每次一次攻击落在尸兄脖子上，都会带来一连串的火花，但所造成的伤害，也只是砍破了一些皮而已。
<br/>　　要知道他的这炳唐刀可不一般，这是他家祖传下来的宝贝，即使放在国际上，在冷兵器中也是极为著名的武器。
<br/>　　唐刀-寒辰。
<br/>　　只要是对冷兵器有所研究的，一定会对这这个名字有所印象。
<br/>　　真不知道，如果换成是普通的武器，现在会发生什么样的事情。
<br/>　　“违规，违规！”
<br/>　　“吃掉，吃掉！”
<br/>　　尸兄的嘶吼着攻来，小胖手持着唐刀，艰难的抵挡着，情势一时间，变的有些危急。
<br/>　　表情有些挣扎，刘哔纵身扑上，死死卡住了其中一名尸兄的脖子。
<br/>　　似乎被他的动作所激怒，尸兄的头颅转了一圈，以一个不可思议的角度，一口咬向了刘哔的脑壳。
