　　第22章
<br/>　　高飞虎不得不承认，虽然纸上的话粗鄙、浅薄，但煽动力极强，就连自己也隐隐有些动心。
<br/>　　而飞虎寨的一帮大老粗们，已经兴致勃勃开始揭树上的白纸。
<br/>　　“老八，那明明是我先看见的，你为什么要和我抢！”
<br/>　　“谁和你抢了，你看见就是你的啊！你明明有匹马，为啥要和我抢！”
<br/>　　“那匹已经老了，老八，你再动手，可就别怪我对你不客气！”
<br/>　　高飞虎无奈让大家把两个要打起来的兄弟拦住，这在山寨中是常有的事情也不算什么。看手下弟兄人人蠢蠢欲动，他也没有办法。
<br/>　　“那么，我们就去看看，看他们耍什么花样。”
<br/>　　于此同时，青牛山各个山寨周围都填满榜单，狗头岭，秃头坡，杏花岭等山寨内，都在议论着这件事。马匹在冷兵器时代，根本就是战略武器，可以直接提升一个山寨的战斗实力。而程大雷亲笔所写的广告词，有些的确戳中他们的心。拥有一匹马有时候话代表着实力、尊严，是身份的象征。
<br/>　　青牛山，蛤蟆寨。
<br/>　　程大雷几个守在寨门前，远远看着上山的山道。
<br/>　　“大当家，你说会有人来么？”徐神机。
<br/>　　“等一等，传播和转化总需要一定时间。”
<br/>　　“虽然你说得我都听不懂，但总感觉好有道理的样子。”徐神机。
<br/>　　黄三元在旁边一激灵，那种不好的感觉再次出现，自己刚认的这位老大，该不会又是一个韩玄之吧。自己怎么这么悲催，总是遇到这样的人……
<br/>　　“来了！”徐灵儿忽然指着山下道。
<br/>　　众人眼睛一亮，就见山道之上，有七八个人正走向这里。是飞虎寨的高飞虎、高飞豹兄弟，不仅仅是他们一伙人，此刻在山道上，有许多人出现，都是各山各寨的土匪，少则两三人，多则十余人。差不多青牛山方圆的山寨都到了。
<br/>　　“快快，大家做好准备，别出了岔子。”
<br/>　　高飞豹还骑着他那匹独眼马，这匹马瞎了一只眼，容易跑偏，一路上总是转圈子。可以说能够成功骑到这里，极其的不容易。
<br/>　　“喂，你们快把门打开，老子来买马。”
<br/>　　“秦蛮，备弓。”程大雷道。
<br/>　　“是！”秦蛮拉弓搭箭，瞄准高飞豹。
<br/>　　高飞豹一激灵，赶忙下马拦在马头前：“怎么，一只眼也不给我留！”
<br/>　　高飞豹轻咳一声，道：“我这兄弟行事鲁莽，程大当家不必动怒，我们是诚心实意来买马，你看粮食我们都带来了，马在哪里，是不是让我们先看一下。”
<br/>　　此刻，各山寨的人差不多已经到齐，聚集在蛤蟆寨的大门前。人人凶神恶煞，狰狞跋扈，嘴里骂骂咧咧。
<br/>　　程大雷见到这一幕，微微皱眉，忽然爆喝道：“我有神法，能请天雷……”
<br/>　　一瞬间，鸦雀无声，这般大老粗都识趣的闭上嘴巴。
<br/>　　“对，这样才乖嘛，黄管家给大家讲讲规矩。”程大雷放下双手，心里也知道这把戏估计糊弄不了多久，必须让山寨尽快的发展起来。
<br/>　　黄三元见程大雷说话这么好使，心里也有了自信，原来自己这个老大这么有面子。
<br/>　　“这次我们是用暗拍的方式。”
<br/>　　“什么是暗拍？”高飞豹。
<br/>　　“这位朋友问了，什么叫暗拍，我正要向大家解释，大家看到门上那块红布没有？”
<br/>　　所有人顺着黄三元手指的方向去看，只见蛤蟆寨的木门上挂着一张红布，将大门遮得严严实实。
<br/>　　“每拍卖一匹马，我们都事先将马牵到大门后，大家各自出价，价高者得。”
<br/>　　“这样岂不是说我们连马是什么样都看不到！”
<br/>　　“那后面要是一只瞎马瘸马呢。”
<br/>　　“你们要是暗自将我们买好的马调换呢。”
<br/>　　群贼有激动起来，黄三元的声音根本压不住场子。
<br/>　　“雷啊……”
<br/>　　群贼闭上了嘴巴，程大雷风轻云淡的看向远处，趁着好用还是多用几次吧。
<br/>　　“我们肯定是不会暗中调换，这红布透着光，后面什么动静大家其实都看得着，只是看不清马什么样而已。至于买到的是好马瘸马，真的就是看大家运气，也许你花极少的价钱买到一匹千里驹，当然，相反的情况也会出现，这就是赌嘛。”
<br/>　　黄三元说着目光转了一圈，见没有人说话，他点点头：“既然大家没有别的意见，拍卖正式开始，有请第一匹马。”
<br/>　　大家透过红布，能看到一匹马被牵到后面，具体马是什么样子，却是没人看到。
<br/>　　“第一匹马，起拍价粮食一石！”黄三元铛地一声敲响手中铜锣。
<br/>　　一帮山贼面面相觑，有些山贼还没弄明白是怎么会事，大家看不到马匹也不敢出价。程大雷盘腿坐在一块大石头上，见到这一幕不免有些打鼓：该不会玩得太高端，玩砸了吧。
<br/>　　“我出一石大豆。”
<br/>　　一个清亮的女人声音响起，程大雷心中一动，竟然还有女山贼。他目光钻过去，只见这女人穿着棉布衣服，头上梳着一条长长的大黑辫子，最显著的特点是白……白得像六月的棉花一样，能看到皮肤下面的毛细血管。
<br/>　　“这位是杏花岭的小白狼。”徐神机悄悄道。
<br/>　　不仅程大雷，其他山贼见了也移不开眼睛，有人还发出咕咚咚咽口水的声音。
<br/>　　“看他娘的什么看，回去看你妈去，再看你姑奶奶把你那玩意割下来喂狗。”小白狼拔出一柄雪亮腰刀，在空中舞了个刀花。
<br/>　　程大雷也觉得裤裆飕飕发凉，忙闭上眼睛，装作呼吸吐纳的样子。
<br/>　　“我们是恭喜白当家，这么便宜就买到一匹好马，心里羡慕呐。”高飞豹捂着嘴笑道。心道：一石大豆能换什么好马，连马鞍都换不到一副。
<br/>　　众贼心中窃窃私语，想的都是同样的事情。
<br/>　　“好，第一匹马就由杏花岭白当家拍得，林少侠牵马。”
<br/>　　虽然群贼心里都不报希望，但还是忍不住好奇心递过去一只眼睛。只见木门缓缓开启，一匹马被牵了出来。
<br/>　　唰！
<br/>　　无数人眼睛都瞪直了，有得甚至站在地上都站不稳。
<br/>　　
<br/>(本章完)
