　　第760章
<br/>　　张天赐就坐在程大雷对面，丰神俊朗，宛若明月大山，光明磊落。他丝毫不避讳提及一枝桃的事，却也丝毫不在意。那个名字对他来说，不过是肩上些微尘埃。随手拍去，些微尘埃怎么影响张天师的神仙气质。
<br/>　　他脸上一直挂着淡淡的笑，但这笑容忽然一凝。若有若无，他感觉到一种不安的气氛，似乎正被某只凶兽注视。但这感觉一闪而逝，他的目光落在程大雷身上。
<br/>　　程大雷一张丑脸笑着，却又极其认真问：“贵庚？”
<br/>　　张天赐不知程大雷为何突然发问，下意识回答道：“虚度光阴二十七载。”
<br/>　　“啧。”程大雷叹了一声，似乎有些遗憾。
<br/>　　张天赐不知他这遗憾从何而来，正要开口询问。却见程大雷已经起身，向玉皇观外走去。看他的背影，并没有回头的打算。
<br/>　　张天赐挠挠头，望着程大雷的背影，浑然不知发生了什么。
<br/>　　程大雷三人离开了玉皇观，来到了山下。
<br/>　　“大当家，咱们现在去哪里？”
<br/>　　“去满庭芳看看。”
<br/>　　百里胜死在满庭芳，以他的身份，官府定已经着手调查。有很大可能，这件事会怀疑到自己身上。除此之外，程大雷也想搞明白，究竟是谁下的手。
<br/>　　三人一路赶到满庭芳，与所料不差，那院落已经被官兵团团围住。几个官员出入其中，像是衙门口的人。
<br/>　　程大雷被挡在了门外，守门官兵黑着一张脸。
<br/>　　“无关人等，速速离开，小心抓你下大牢。”
<br/>　　程大雷还未开口，就听刘发财喝了一声：“瞎了你的狗眼，不知道这是谁么。程当家过来，你也敢拦？”
<br/>　　守卫一激灵，弄明白程大雷的身份后，竟下意识向后退了一步。
<br/>　　长安城只有一个大当家的，那便是恶贯满盈的程当家。长安城大小人物都是对程大雷如此称呼，一来，现在他的身份有些尴尬，非官非；。二来，大家明面上不能将他怎样，只好从心底鄙视他的山贼出身。
<br/>　　守卫有些为难，上面发话不能放任何人进去，这也不是自己能做主的。况且，百里胜是刚和程大雷见过面之后死的，主要疑点都在他身上。
<br/>　　现在他突然过来，是自投罗网，还是有恃无恐。
<br/>　　“程大雷在哪里，程大雷在哪里？”
<br/>　　突然从院内闯出十几条壮汉，盯着程大雷的眼睛恨不得将他生吞活剥。这群人都是跟着百里胜从凉州一路杀过来的，对百里胜自然有很深的感情。
<br/>　　当初都是小兵小卒，如今至少都混上了校尉的官职。明日之后，他们会成为帝国军中的中坚力量。
<br/>　　“恶贼，你还敢来！”
<br/>　　“弟兄们，动手。”
<br/>　　十几个人同时拔出兵器，哗啦啦刀响，向程大雷扑了过来。
<br/>　　程大雷无可奈何的撇撇嘴，右腿向前迈了半步。
<br/>　　拳打脚踢，眨眼间程大雷便放倒一片。十几个军中悍将，没有一人能进的了程大雷的身。
<br/>　　这群人被揍得鼻青脸肿，有人咬着牙，从嘴角渗出血来。看向程大雷的眼神难免有些畏惧：此人当真不好对付。
<br/>　　程大雷目光冷冷扫过：“百里胜死后就没人管得了你们不成。要知道，百里胜活着时，也不敢在本当家面前放肆。”
<br/>　　这时候，有一人从院门走出，挥挥手喝令诸人退下，然后抱拳向程大雷行了一礼。
<br/>　　“在下卢俊义，见过程当家。程当家请……”
<br/>　　程大雷看向卢俊义的眼神有些无奈，叹口气，随着卢俊义走入院中。
<br/>　　院内还有战斗过的痕迹，事发时的茶室更加狼狈，地上落满血。
<br/>　　“查出什么没有？”程大雷问。
<br/>　　卢俊义回答道：“衙门的捕快已经仔细验过，确认动手的有十三人。他们先用弓箭堵住院门，之后再一拥而上。侯爷手中没有趁手的兵器，遭了他们的毒手。”
<br/>　　程大雷仔细看着茶室内的景象，其实也看不出什么。他没有查案的经验，如果宋慈在此地，应该会剥丝抽茧，寻出背后真凶。
<br/>　　但程大雷却也不是宋慈，他道：“对方怎么走的，可以好好查一查？”
<br/>　　卢俊义点点头。
<br/>　　十几名刺客能在长安城突然消失，必然有大人物出手遮掩，否则谁也无法把事情做得如此干净。
<br/>　　看来百里胜拔地而起，他如日中天的势头挡了不少人的路啊。
<br/>　　“已经派人去查了，方圆五里之内仔细搜索。”卢俊义道：“我冒昧想问一句，是否是程当家动的手？”
<br/>　　程大雷无奈看了他一眼，道：“杀了他于我何利？”
<br/>　　“我相信程当家。”
<br/>　　虽然山贼这个职业，总是被人觉得对杀人有某种偏爱。这件事大家怀疑程大雷也算是正常，但按照正常人的想法，程大雷的确没有必须杀百里胜的理由。
<br/>　　程大雷觉得，这件事查也不会查出什么线索。满庭芳周围，方圆多的是王公贵族，他们的宅子是想搜就可以搜的么。如果真是有大人物遮掩，想要隐藏这伙人也不是太困难的事情。
<br/>　　怕这件事最终也不会查出什么。程大雷叹了口气，目光落在卢俊义身上，心中忽然一动。
<br/>　　“卢将，侯爷已经死了，您给为自己谋个出身呐。我凉州兵多将广，到那里定会做出一番大事业，卢将军是否考虑一下？”
<br/>　　卢俊义明显因程大雷的话震住了。虽然说我暂时相信你，但百里胜刚死，你就开始挖墙角，这事情做得也忒不地道了吧。
<br/>　　他深吸一口气，压抑住心中的怒气，严肃道：“侯爷对我恩重如山，如今他身遭横死，人头被人砍去，我必找到真凶，为侯爷复仇。”
<br/>　　最后，他看了一眼程大雷：“这件事最好不要和程当家有关系，否则，我定要向程当家讨个公道。”
<br/>　　最后，他还是点了点程大雷。
<br/>　　程大雷一惊：“他的人头被割走了？”
<br/>　　卢俊义点点头。
<br/>　　程大雷眉头皱了起来。如果说，只是因为百里胜挡了谁的路，让他死就足够了。可杀死他都不肯给他一个全尸……
<br/>　　这件事怕还藏着一些别的东西。
<br/>(本章完)
